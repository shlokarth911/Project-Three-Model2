Built in ThreeJS

Three.js is a popular JavaScript library that makes it easy to create 3D graphics in the browser using WebGL. It provides a straightforward API to build and animate complex 3D scenes, making it accessible even for those new to 3D development. With Three.js, developers can add objects, textures, lights, and shadows, allowing for immersive visuals and interactive experiences. Its versatility supports applications in gaming, data visualization, virtual reality, and interactive websites, enhancing user engagement and visual appeal.
